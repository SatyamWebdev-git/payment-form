# Payment Form

This is a simple and responsive payment form built using **HTML** and **CSS**. It allows users to enter basic payment details such as name, card number, expiry date, and CVV.

## Features
- Clean and modern user interface
- Responsive layout for mobile and desktop devices
- Placeholder hints for user-friendly input

## How to Use
1. Download or clone the repository.
2. Open `index.html` in any browser.
3. Fill in the form fields and explore the layout.

## Technologies Used
- HTML
- CSS
